
## Description
**Wedding Invite Generator**
This repository features a ReactJS-based wedding invite generator, enabling users to easily create and customize digital invitations for guests. Built with React, HTML, CSS, and Vite, it offers a simple and interactive solution for designing and sharing wedding invites online.<br/>
